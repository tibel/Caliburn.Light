# Security Policy

## Supported versions
Only the **latest** version is being supported with security updates.

## Reporting a vulnerability
Please do not report security vulnerabilities through public GitHub issues.

Instead, please report them using [GitHub's private vulnerability reporting](https://github.com/tibel/Caliburn.Light/security/advisories/new).

## Security advisories
The project team is committed to transparency in the security issue disclosure process. We announce security issues via project Github Release notes.
